#### 📝  介绍 
    前后分离,后台通配权限管理系统！

    数据库脚本位置根目录 doc 文件夹

    后端技术：.Net6、EFCore6、Swagger 、Redis
    
    前端：Vue3.x 、Antd Of Vue 3.0 、Vite
    
    开发需要环境：vs2022 、 .Net6.0 、VsCode 、Redis
