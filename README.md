### QQ交流群

### 📝  介绍 
    前后分离,后台通配权限管理系统！

    数据库脚本位置根目录 doc 文件夹 目前仅提供了 MySql、SqlServer、PgSql 脚本 ，其他请使用efcore迁移（迁移教程请看语雀文档）

    后端技术：.Net6、EFCore6、Swagger 、Redis
    
    前端：Vue3.x 、Antd Of Vue 3.0 、Vite 、Pinia 2.0.x 、TypeScript
    
    开发需要环境：vs2022 、 .Net6.0 、VsCode 、Redis
